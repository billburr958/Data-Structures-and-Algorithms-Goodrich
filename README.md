## Solutions Included

Each chapter directory contains:
- **solutions.py**: Implementations of all exercises (Reinforcement, Creativity, and Projects)
- **tests/**: Unit tests and test runners for the solutions
- **README.md**: Chapter-specific notes and explanations

## Exercise Categories

- **Reinforcement (R-x.y)**: Basic exercises to reinforce chapter concepts
- **Creativity (C-x.y)**: Problems requiring creative application of concepts
- **Projects (P-x.y)**: Larger programming projects integrating multiple concepts

## Running the Code

### Prerequisites
- Python 3.7 or higher

### Running Solutions
Each chapter's solutions can be imported and used individually:

```python
from chapter_01.solutions import is_multiple, minmax, norm

# Test a specific function
result = is_multiple(10, 5)  # Returns True
print(result)
```

### Running Tests
Navigate to a specific chapter and run the test files:

```bash
cd chapter_01/tests
python test_runner.py
```

## Code Style
